# Security and Sensitive Data

Do not report or attach real participant video, audio, images, biometric templates, exported labels, or private medical context in public issues.

For security concerns, contact Genesis Lab Foundation privately before publishing details.

## Sensitive Data Rules

- Do not commit `.env` files, recordings, snapshots, or label exports.
- Do not add remote analytics or background upload logic.
- Do not add biometric identity matching without explicit ethical review.
- Keep participant examples synthetic, public-domain, or explicitly consented.
